01 Workbench Origin

HTTPS 固定 · https://DOMAIN ONLY
HTTPS必須 — 暗号化された接続—
ドメイン名FQDN が必要（例 host.example.ts.net）—
IP アドレス直打ちは不可 — 証明書が効かない—

公式拡張は HTTPS ドメインでのみ動作。tailnet は MagicDNS 名（例 xxx.ts.net）を使用。

02 最初に開くフォルダ

📁
/home/user

サーバに接続すると、フォルダを参照して選べます。手入力はいつでも可能です。
